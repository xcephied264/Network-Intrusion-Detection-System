# Network-Intrusion-Detection-System
Developed a machine learning-based intrusion detection system using the KDD dataset
Implemented and compared multiple models, including Linear SVM and Random Forest
Applied preprocessing techniques such as one-hot encoding and feature scaling to optimise model input
Performed hyperparameter tuning to improve model performance and generalisation
Evaluated models using classification metrics including accuracy, precision, recall, and F1-score
Achieved improved performance after tuning, with the SVM model reaching 98% accuracy
Reduced false positives and improved detection rate compared to the untuned baseline model
